# esupa_store_pos

An POS app

## Getting Started

**Install Flutter:**



### Using Mason

**Activate Mason brick:**
```shell
dart pub global activate mason_cli
```
```shell
cd project
mason get
```
**Create new feature folder:**
```shell
# path to feature
cd lib/**/features/
```

```shell
# put space at the end of the command
mason make feature
```

#### Updating Mason Bricks

```shell
mason cache clear
```
```shell
mason get
```
